Typography

Company Name:

Zillow

How Used:

Typography in design is about the creation and flow of text in design. When good typography is part of your design, you understand the importance of hierarchy. It’s the way a user sees items on a page. What is that they see first? Do you want that message to be seen first? The Zillow Mobile App typography allows the user to see the slogan “Reimagine home” first with the logo right about it in almost the same bold font. This let’s the user know that Zillow can help the user “Reimagine Home”.
